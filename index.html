THE ACRE DEAL - LEAD SOURCING FRAMEWORK

PURPOSE

This framework provides Land Intake Managers (LIMs) with a systematic approach to source qualified land seller leads through effective phone conversations.

GOAL

The following 7-phase script is designed to naturally uncover these three criteria through conversational flow:

  • Phase 1: Opening, Interest Check (IF) & 5 min Permission Check
  • Phase 2: Uncover Motivation (WHY) & Timeline (WHEN)
  • Phase 3: Set Price Expectations
  • Phase 4: Ownership Verification
  • Phase 5: Access Verification
  • Phase 6: Additional property details
  • Phase 7: Summarize & Close
